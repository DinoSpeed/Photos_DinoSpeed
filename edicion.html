    PROGRAMAS PARA EDICION DE FOTOGRAFÍAS
    PHOTOSHOP
      Photoshop es una herramienta profesional que se usa principalmente por fotógrafos y diseñadores
    		PHOTOSHOP
      Entre sus características se encuentran:
    • Filtros: Añadir efectos como desenfoque de movimiento, acuarela, posterización, mosaico, entre otros
    • Efectos especiales: Añadir luces, efectos de iluminación, y texturas
    • Diseño web: Crear diseños para páginas web
    • Animaciones: Crear animaciones en 3D, aplicar efectos de movimiento, y más
    • Edición de videos: Editar videos, crear clips, y aplicar efectos de movimiento
    • Edición de texto: Calcular el tamaño de fuente predeterminado en función de la resolución del documento
    • Fotomontaje: Realizar técnicas de fotomontaje
    • Ajustes de imagen: Ajustar brillo, contraste, saturación, recorte, rotación, y tamaño de lienzo
    • Creación de mapas de bits y GIFs: Crear mapas de bits y GIFs
    • Modificación de colores: Modificar colores, añadir figuras y formas
    RAWTHERAPEE
      Algunas de las funciones de RawTherapee son: Etiquetar, Calificar, Filtrar, Enmascarar el área recortada de la imagen, Guardar los parámetros en un archivo sidecar independiente.
    		RAWTHERAPEE
      RawTherapee es un programa de edición de imágenes RAW que tiene varias características, entre ellas:
    • Edición no destructiva: Los ajustes se reflejan en la imagen de vista previa, pero no se aplican a la imagen abierta
    • Panel del Historial: Muestra un listado de todos los cambios realizados en la imagen
    • Apertura de varias imágenes: Permite abrir varias imágenes al mismo tiempo
    • Filtros y algoritmos: Cuenta con diversos filtros y algoritmos de interpolación cromática
    • Interfaz gráfica: Permite un uso más eficiente del espacio en pantalla
    • Licencia: Se publica bajo la Licencia Pública General GNU versión 3
    • Multiplataforma: Funciona en distintas plataformas
    • Base DCRAW: Utiliza DCRAW, una aplicación de código abierto de revelado
    LIGHTROOM
      Adobe Lightroom es un programa que permite editar, organizar, almacenar y compartir fotos. Está disponible para ordenadores, smartphones y tabletas.
    		LIGHTROOM
      Características:
    • Permite ajustar la exposición, el color, la claridad y añadir viñetas
    • Incluye herramientas de edición diseñadas para computadoras
    • Permite organizar las fotos por palabras clave
    • Permite revelar imágenes en formato RAW
    • Permite exportar las fotos a formatos como JPEG o TIFF
    • Permite almacenar los archivos en la nube
    • Permite acceder a los archivos desde cualquier dispositivo
    • Permite personalizar la configuración predeterminada de las fotos RAW
    • Permite añadir marcas de agua de texto a las fotos
    • Permite ajustar el tono local
      Requisitos de Lightroom:
    • Los requisitos de sistema de Lightroom varían según el sistema operativo y la versión del programa
      • >
    • Se requiere una conexión a internet para activar el software y validar los abonos
    • Se requiere espacio en el disco duro para instalar el programa

      • Compatibilidad de Lightroom:
    • Lightroom para iOS es compatible con iPhone y iPad que ejecuten iOS 17.0 o posterior
    AFFINITY PHOTO
      software de edición fotográfica que ofrece funciones para retoque, revelado RAW, composición, selección y más.
    		AFFINITY PHOTO
      Características:
      • Capas de ajuste
    • Permite ajustar el brillo, el contraste y las curvas
    • Permite agrupar, recortar, enmascarar o fusionar efectos y filtros
      • Pinceles
    • Cuenta con una gran biblioteca de pinceles, incluidos los personalizados
    • Permite pintar de forma natural
    • Permite combinar varios pinceles en un solo trazo
      • Efectos y filtros
    • Cuenta con una gran biblioteca de efectos y filtros en tiempo real
    • Permite crear composiciones de imágenes complejas
      • Revelado RAW
    • Permite hacer fotografías en formato RAW y disfrutar de una experiencia integral de 32 bits
    • Cuenta con un motor SerifLabs RAW actualizado
      • Panorámica
    • Cuenta con una función de unión panorámica que permite fusionar fotos para crear una escena más amplia
      • Compatibilidad
    • Está disponible para todas las plataformas, incluido el iPad
    • Es compatible con imágenes muy grandes
    • Permite cargar casi instantáneamente archivos de gran tamaño
      • Otras características
    • Permite cambiar el tamaño, recortar y enderezar las fotos
    • Permite hacer selecciones muy precisas
